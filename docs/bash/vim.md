# VIM

## Modes (3 main modes)
- Normal
- Insert
- Visual

## Commands
C I )
- C - Change
- I - Inside
- ) - Parens

C I " Change inside Quotes
C I } Change inside Brackets
D I ) Delete inside Parens

[Cheat Sheet](http://www.viemu.com/a_vi_vim_graphical_cheat_sheet_tutorial.html)
